# ♿ 0.6m Viewpoint - IoT 이동약자 3D 시뮬레이터

> **"0.6m의 낮은 안구 높이에서 마주하는 일상의 장벽과 보행 장애 환경의 공감"**
>
> 본 프로젝트는 비장애인이 3D 가상 서점 공간에서 수동 휠체어 사용자의 신체적 제약과 인지부하를 직접 체험함으로써, 이동약자의 시선에서 마주하는 사회적 장벽과 배리어프리(Barrier-Free) 설계의 중요성을 체감하도록 돕는 디지털 시뮬레이터입니다.

---

## 🎮 시뮬레이터 미리보기 & 실행 방법

### 1. 개발 서버 실행 방법
본 프로젝트는 별도의 프레임워크 설치 없이 경량 Node.js 파일 서빙 시스템으로 구동 가능합니다.
```bash
# 1. 저장소 클론
git clone https://github.com/minulx-10/barrierfree-3d-simulator.git
cd barrierfree-3d-simulator

# 2. 로컬 개발 서버 구동
node server.mjs
```
- 서버가 실행되면 웹 브라우저에서 **`http://127.0.0.1:4175`**로 접속하여 즉시 시뮬레이션 및 소개 페이지를 체험할 수 있습니다.

### 2. 조작 가이드
* **휠체어 조향 및 이동**: `W`, `A`, `S`, `D` 키 (실제 휠체어의 묵직한 회전 관성과 가속 지연이 반영되어 있습니다.)
* **시선 방향 조절**: `Mouse Drag` (휠체어 눈높이 0.6m에서 시점을 둘러봅니다.)
* **수동문 개방 / 상호작용**: `Space` 키 연타 (무거운 통유리문 앞에서 반복해서 눌러 저항을 이겨내고 문을 엽니다.)

## ⚡ 주요 고도화 기능

### 1. 휠체어 전용 물리 엔진
- **카메라 시점 낮춤 (1.62m ➡️ 0.60m)**: 휠체어 착석 시의 제한된 수직 시야각과 공간 왜곡 모사.
- **물리 모델 및 관성**: 방향 전환 지연 속도 적용 및 선형 속도 부드러운 감쇠(Velocity Lerp)로 묵직한 조작감 구현.

### 2. 피로도(Stamina) 시스템
- 주행 거리에 따라 스태미나가 소모되며, **경사로 및 단차(문턱)** 진입 시 평지 대비 **3.5배** 빠르게 소모됩니다.
- 피로도가 20%를 초과할 경우 신체 제약으로 인해 주행 최대 속도가 점진적으로 **최대 40% 감속**됩니다.

### 3. 심리적 불안감(Anxiety) 시스템
- 움직이는 시민(NPC)이나 도서 책장, 벽 등에 초밀접(1.8m 이내)하게 접근하면 불안감 수치가 점증합니다.
- 불안감 크기에 비례하여 화면 주변부가 흐려지는 **시각 흐림(Blur) 필터**와 고조되는 **카메라 셰이크(Camera Shake)**가 연출됩니다.

### 4. 2개 영역의 스마트 실시간 HUD
- **좌측 페르소나 HUD**: 휠체어 사용자의 현재 좌표, 주행 속도, 누적 충돌 횟수, 그리고 관절 내구도(Stamina) 게이지 실시간 피드백.
- **우측 실시간 상태 패널**: 물리적/심리적 수치 모니터링 및 3단계 위험도 인지 메시지 안내 (`[안정]` ➡️ `[상황인지]` ➡️ `[판단/수행]`).

### 5. 다이나믹 결과 모달 & 다시 도전
- 미션이 종료되면 총 소요 시간, 스트레스 수치, 부딪힌 횟수, 장애물 우회 성공 횟수가 요약된 리포트 카드가 출력됩니다.
- 실패 조건을 통과하지 못했거나 재도전하고 싶을 때, 모든 물리 수치와 도어 및 NPC 위치를 초기화하여 바로 다시 체험할 수 있습니다.

## 🧱 체험하는 5가지 접근성 장벽 (Obstacles)
1. **고중량 수동 통유리문**: 문 앞에서 양팔로 휠체어 바퀴를 잡은 채 무거운 문을 밀며 들어가야 하는 저항감을 구현.
2. **좁은 서가 구간**: 표준 휠체어 폭(70cm)과 거의 대등한 80cm 동선에서 책장을 부딪치지 않고 빠져나가야 하는 좌우 조향의 한계.
3. **혼잡한 시민 동선 (NPC)**: 주변을 빠르게 스쳐 지나가는 무작위 보행자들과의 속도 격차가 유발하는 심리적 위험 요인.
4. **높은 도서 검색대 (바 체어)**: 입식 높이(1.2m)로 제작된 가구들이 앉은 키 시점에서 주는 단절감 체감.
5. **핸드레일 없는 계단 & 우회 경사로**: 낙상 위험이 극대화되는 난간 없는 계단을 마주하고, 3.5배의 피로도 감수가 수반되는 가파른 오르막 경사로로 안전하게 우회하는 선택지 마련.

## 🛠 기술 스택 (Tech Stack)
- **Frontend Core**: Vanilla HTML5, Vanilla JavaScript (ES6+ Module)
- **3D Render & Collision Engine**: Three.js (r125/r165), Cannon-es (CANNON.HingeConstraint 물리 연동)
- **Styling & Layout**: Vanilla CSS (CSS Variables), Tailwind CSS v3
- **Development Server**: Light-weight Node.js HTTP Server (`server.mjs`)
